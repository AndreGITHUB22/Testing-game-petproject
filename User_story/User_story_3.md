ID: US-03<br>
Category: Combat / Equipment<br>
User Story:<br>
As a player, I want to equip and unequip weapons, so that I can use them when needed.<br>
Acceptance Criteria:<br>
1)Player can equip a weapon from inventory<br>
2)Player can switch between weapons<br>
3)Player can unequip (hide) the current weapon<br>
4)Player cannot equip a weapon if inventory is empty<br>
