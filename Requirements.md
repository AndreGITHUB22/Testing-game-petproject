# Requirements — The Precinct Testing

## 1. Overview
This document describes high-level requirements for gameplay and AI behavior.

---

## 2. Functional Requirements

### 2.1 Callout System
- The game should generate random callouts
- The player should be able to accept or ignore them

### 2.2 AI Behavior
- NPCs should react to player actions
- Suspects should attempt to escape during pursuits
- Partner AI should follow the player without getting stuck

---

## 3. Non-Functional Requirements

### 3.1 Performance
- The game should maintain stable FPS on Steam Deck

### 3.2 Usability
- UI prompts should be clear and understandable

---

## 4. Assumptions
- The game runs via Proton on Steam Deck
